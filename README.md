# run pseudocode.js code

## Notice
Node.js v8.9.0+ required


## Build
build with webpack, produce `public/RunPseudoCode.bundle.js`

## how to
run in Node.js
1. yarn
2. yarn dev
3. `modify code in index.js`

or view on Browser
1. yarn install -g serve
2. serve public
3. open `http://localhost:5000` (or URL on terminal)
